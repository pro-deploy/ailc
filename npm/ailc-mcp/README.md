# ailc-mcp

Обёртка для запуска [ailc](https://github.com/pro-deploy/ailc) (AI Life Cycle) через `npx`. При первом запуске скачивает готовый бинарь для вашей платформы из релиза GitHub, кэширует его и запускает. Зависимостей нет.

ailc это автономный оркестратор качества и безопасности кода: детерминированный гейт (гарантию даёт гейт, не нейросеть), глубокий межпроцедурный анализ заражения по пятнадцати языкам, статический разбор по дереву, правила OWASP и Web/API, вывод SARIF, сигналы соответствия требованиям РФ. Один офлайн-бинарь на Rust.

## Подключение в среду разработки

В `.mcp.json` (Claude Code) или в `~/.cursor/mcp.json` (Cursor):

```json
{ "mcpServers": { "ailc": { "command": "npx", "args": ["-y", "ailc-mcp"] } } }
```

Без аргументов обёртка запускает MCP-сервер (`ailc serve`). Можно также вызвать любую подкоманду, например `npx -y ailc-mcp dod .` или `npx -y ailc-mcp pulse .`.

Поддержаны Linux (x86_64 и aarch64), macOS (Intel и Apple Silicon), Windows (x86_64). Бинарь кэшируется в `~/.ailc/bin/`. Полная документация в [репозитории](https://github.com/pro-deploy/ailc).

Лицензия Apache-2.0.
